## 📌 Overview  
This is a **full-stack MERN** (**MongoDB, Express, React, Node.js, TypeScript**) e-commerce application with:  
- **User authentication** (JWT access & refresh tokens)  
- **Product management**  
- **Cart functionality**  
- **Best practices & modular, reusable code**
